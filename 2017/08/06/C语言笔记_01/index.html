C语言笔记_01

发表于 2017-08-06 | 分类于 C语言

关键字

c语言中所有关键字（32个）都是小写

标识符

(自己起得名字)
字母（大小写），数字，下划线（不能以数字开头）

注释

//:单行注释
/**/:多行注释
单行注释可以嵌套多行注释(次多行注释必须在同一行)
多行注释，可以嵌套单行注释

数据类型

基本数据类型

整型：短整型(short)，整型(int)，长整型(long)
字符型：(char)
(实型)浮点型:float(只保证7位数字有效),double（默认为双精度）

构造类型

数组，结构体，枚举，共用体
指针类型
空类型(Void)

常量

整型常量 10
实型常量 10.1（双精度） 10.1f（单精度）
字符型常量 ‘a’ （只能一个字符）错误实例：‘南’，因为一个汉字占用三个字节。
特殊情况：系统保留给某些字符的组合定义了特殊含义，这些字符的组合可以当做单个字符（例如：\n）
字符串常量 “abc”

变量

定义变量，但是没有初始化，内部存储的是垃圾数据，不同平台这个数据可能不同
int num;
printf(“%i\n”,num);//1606422582
解析：%代表占位符，i代表int类型

部分初始化

int num,num1=10;
解析：定义了两个变量num,num1,并且给num1进行初始化为10

完全初始化

int num,num1;
num=num1=10;
解析：都初始化为10

内存分析

类型   16位编译器   32位编译器   64位编译器
 char    1           1           1
 int     2           4           4
 float   4           4           4
 double  8           8           8
 short   2           2           2
 long    4           4           8
 long long 8         8           8
 void    2           4           8

内存寻址是从大到小，越先定义其内存地址越大

int num=0;
int num1=0;
printf(“%p\n”,&num);//0x7fff5fbff838
printf(“%p\n”,&num1);//0x7fff5fbff834
解析：可知num先定义，其内存地址大，而且此时mac系统为64位编译器
int占四个字节，可知内存地址差值也为4.其中&变量名代表该变量的内存地址，
打印内存地址的占位符要用%p。

函数

printf函数

printf(“%d,\n%d\n”,num,num1);
注意：double
double num=0.1;
printf(“%lf”,num);

指定位宽（占多少列，例如99就是两列）默认情况下为右对齐

int num=11;
printf("%5i",num);
输出：   11，说明：前面空出三列
printf("%-5i",num);
说明：此时对齐方式为左对齐
如果:实际输出超出位宽，则原样输出
int num=1111;
printf("%2i",num);//1111
前面补充0的案例：
int num=1;
printf("%02i",num);//01
说明：此时实际输出小于位宽，前面补空格，但是此时位宽前面有0，则补充空格变化为补充0

保留位数

float num=3.14;
printf(“%.3f\n”,num);
解析：%f默认保存6位小数，如果利用.n可以指定保存多少位小数
注意：float有效位数(从第一位数字开始，包括小数点)是7位，超出之后，不论怎么设置都不准确
例如：
float num=3.141592678;
printf(“%.9f\n”,num);//3.141592741
如果想完整输出该数字，必须将该数据保存为double类型，doule有效位数为15位
提高逼格的案例：
float num=3.141592678;
printf(“%.f\n”,5,num);//3.14159
printf(“%.f\n”,2,num);//3.14

scanf函数

int num;
scanf("%i",&num);//必须告诉scanf函数变量的详细地址，才能存储用户输入的变量
printf("%i\n",num);
案例：计算两个输入值的和
int num,num1;
scanf("%i,%i",&num,&num1);
printf("%i\n",num+num1);
说明：输入也要对应格式化字符串中的样式，例如此时可输入123,123
如果%i-%i则输入也要用123-123,如果格式化字符串中没有指明，-这些分割，
在输入时可使用空格回车等实现多次输入(但是%c除外，因为空
格等也会被当做字符被char接收，所以最好都加上逗号这种做分割，避免使用没有分割时空格赋值给char造成错误)。

格式化字符中类型列表

d   十进制整数
0   八进制整数
x   十六进制整数
i   有符号或无符号的十进制，八进制，十六进制整数
u   无符号整数
f   浮点数
c   字符
s   字符串
p   地址值

运算符

取模运算符号取决于左边的操作数

案例：
int num=10 % -3;
printf(“%d”,num);//1
取模运算左边小于右边，输出左边
int num=1 % -3;
printf(“%d”,num);//1

隐式转换和强制类型转换

int num=10.8;
printf(“%d\n”,num);//10
int num=(int)10.8;
printf(“%d\n”,num);//10

算术运算注意：参与运算操作数是什么类型，结果就是什么类型

int num=1/2;
printf(“%d\n”,num);//0
c语言中也有-=,*=,/=,+=，++，–等

sizeof运算符：计算一个常量，变量，一种数据类型在内存中所占的字节数

注意：sizeof是运算符，不是函数
int num=1/2;
int size=sizeof(num);
printf(“%d\n”,size);//4

逗号运算符:常用于简化代码

    int a=15;
    int b=5;
    int result;
    a+=10,b+=1,result=a+b;
    printf("%d,%d,%d\n",a,b,result);//25,6,31
注意：只要是表达式都有结果，逗号表达式也不例外
    int a=15;
    int b=5;
    int result=((a+=1),(b+=1));
    printf("%d\n",result);//6
解析：逗号表达式结果就是最后一个表达式的结果。

关系运算符：在C语言中，只有0才为假

关系运算符中：>,<,>=,<=优先级大于==,!=
关系运算符返回值只有两种：0（假），1(真)
案例：
int result=10>1==1;
printf("%d\n",result);//1
注意：算术运算符优先级大于关系运算符
 int result=2+2>1+1;
 printf("%d\n",result);//1
如果优先级和结合性同时存在，先优先级在结合性（左结合性和右结合性，此时为左结合性）
int result=2+2>1+1==3>1;
printf("%d\n",result);//1
右结合性案例：
a=b=c;

逻辑运算符:返回值只有两个：1，0

&&:逻辑与，结合性，从左至右
||:逻辑或，结合性，从左至右
!:逻辑非，结合性，从右至左
int result=0 && 11;
printf("%d\n",result);//0

三目运算符:结合性从左至右，优先级低于关系运算符

int result=10>9?10:9;
printf(“%d\n”,result);//10

流程控制

if语句

 int result=10;
    if(0){
        printf("%d\n",result);
    }else{
        printf("执行了else条件");//此处执行
    }
也有else if这种情况，单独if也可以。

switch语句

其中default后break可省略，因为其后就是结束的大括号。
但是当default部位于最后的时候，break最好别省略，不然其后的都会执行，直到遇到下一个break。
switch的条件表达式必须返回整数的表达式，也就是说()内部只能是整数
注意：char可以转换为int类型，所以()内部也可以放入char。

分号空语句

例如：if();while();for();都代表空语句，也就是说后面代码和这些已经没关系，语句已经结束
for循环默认为真
for(;;)
{
    printf("测试");
}
另外：for语句最简单死循环
for(;;)；
