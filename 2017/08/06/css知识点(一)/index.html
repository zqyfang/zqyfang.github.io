选择器

基本选择器

id(id唯一),元素(div..)，类选择器
        #id        div        .class
        优先级：id>class>元素
        选中的情况下依次根据优先级进行比较，没有选中根据就近原则
        就近原则：没有选中情况下，谁进根据谁显示，选中情况下，如果id等比较也一模一样，会再次根据就近原则处理。
        font-size:60px !important;
        只能设置给属性，不能设置给选择器，无法提升继承的权重。比如给div设置color!important，但是内部p也设置文字颜色，还是按照p的颜色，因为0无限提升还是0.

盒模型

盒子的宽度和高度：
盒子宽度=width(内容宽度)+border(边框)*2+padding(内边距)。
也就代表着padding也会有背景色。
margin：外边距
padding:30px 20px 40px 100px;
遵循：上、右、下、左
还有元素，默认带有padding，比如ul标签，为了方便清除

  • border-width:10px; → 边框宽度
  • border-style:solid; → 线型
  • border-color:red; → 颜色。
  • 等价于：border:10px solid red;

标准文档流

1）空白折叠现象：让img标签之间没有空隙，必须紧密连接。
2）高矮不齐，底边对齐：例如图片大小不一时候，底边是一致的
3）自动换行，一行写不满，换行写。

一、块级元素和行内元素
块级元素对应容器级标签(p)
行内元素对应文本级标签(p除外)
二、块级元素和行内元素的相互转换
display: inline;//block
对比：块级元素默认整行占有，才能设置宽高，行内元素只能靠内容撑出宽高，所以不能设置宽高，但是通过以上可以相互转换。

浮动(脱标)

浮动的元素脱标

注意：盒子a脱标，b还在标准流，两个会相互重叠，层次结构是b在下面，浮动浮动，浮起来在上面。
解析：span是行内元素，无法设置宽高，但是浮动之后就可以设置宽高。

浮动的元素互相贴靠

如果有足够空间，那么就会靠着2哥。如果没有足够的空间，那么会靠着1号大哥。
如果没有足够的空间靠着1号大哥，自己去贴左墙。

字围现象

div挡住了p，但是p中的文字不会被挡住，形成“字围”效果。
