C语言笔记_04(指针相关)

指针基本概念

指针就是用来专门(只能存储地址)保存地址的
定义指针变量：数据类型 *变量名称.
int *p;
p=&num；
在64位编译器，指针占用8个字节

注意点：
1.指针只能保存地址
2.同一个变量可以有多个指针指向它
3.指针的指向可以修改
4.不要访问野指针
5.指针类型是什么类型，就只能指向什么类型的数据

多级指针

指针类型

指针为什么要有类型？
系统会自动根据指针的类型来确定应该取多少个字节的值

指针和数组

指针+1：其实是加上指针类型所占用长度，例如int类型就是加4

指针和字符串

指针保存字符串：char *name="qiang";
//通过指针保存一个字符串，其实保存的就是字符串第0个元素的地址，也是字符串的地址
    char *name="qiang";
    printf("%p\n",&name[0]);//0x100000fa2
    printf("%p\n",name);//0x100000fa2

指针保存字符串和数组保存字符串区别：
数组保存字符串是一个变量，可以修改，存储在栈中，作用域结束会被释放
指针保存字符串是一个常量，不可修改，存储在常量区中，不会被释放，而且多个相同值对应地址相同。

指针数组

指针变量作用

案例：
void change(int *arr){//此处传递过来是数组名，又因为数组名就是地址，所以也可以使用指针接收。
    arr[0]=100;
}
int main(int argc,const char *argv[]) {
    int arr[]={1,2,3,4};
    change(arr);
    printf("%d\n",arr[0]);
    return 0;
}
案例二：
char *get1(){
    char *name="lnj";
    return name;
}
char *get2(){
    char name[]="lnj";
    return name;
}
int main(int argc,const char *argv[]) {
    char *v1=get1();
    char *v2=get2();
    printf("%s",v1);//输出lnj，此时v1是指针保存字符串，存储在常量区中不会被销毁
    printf("%s",v2);//输出为空，此时v2是数组保存字符串，存储在栈中，即使返回了地址，但是栈中字符串随大括号结束被释放，地址已经无效了。
    return 0;
}
案例三：实现strlen函数
int myStrlen(char *str){
    int count=0;
    while(*str++){//因为如果到了最后\0的部分，\0字符的ascii编码是0，C语言中0就是代表假。
        count++;
    }
    return count;
}

函数指针

函数名就是函数的地址
指向函数的指针的定义格式：
void (*指针名称)()：没有返回值没有参数的函数指针
int (*sum)(int ,int)
