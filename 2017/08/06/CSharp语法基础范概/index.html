CSharp语法基础范概

发表于 2017-08-06 | 分类于 .Net

注释

单行注释：//
多行注释： /**/
文档注释：///

快捷键

格式化：ctrl+k（松开k快速按下d）
智能提示:ctrl+j
注释所选代码：ctrl+k+c(k松开)
取消注释所选代码:ctrl+k+u(k松开)
选中当前行：shift+end(如果光标在行首)，shift+home(如果光标在行尾)，如果选中之后，使用上下键
可以选中多行。
折叠冗余代码：在要折叠代码之前(上一行)写上#region,在最后写上#endregion

变量

int   有符号整数
double 64位浮点数    精度小数点后15到16位
float 单精度，定义时候需要加上f，因为浮点数默认为double类型
char 字符，不能存储空字符 char s='';会报错
string 字符串,可以存储空字符串
注意：string是c#独有的，String也等效，但是是.net平台其他语言包括c#都可以用的一个类，
string最后也是映射为String。
decimal 128位数据类型，精度是28到29有效位,常用于财务货币计算
和double相比，有更高的精度和更小(decimal范围更小)的范围
例子：decimal num=1000m   最后必须加上m

命名规则：字母，下划线，或@符号开头，不能以数字开头，后面可以跟任意字母，数字，下划线

运算符

+：只有两边至少有一边为字符串时候就是说起连接eider作用，否则是相加的作用
占位符：其中{0}等就是占位符，替代占位符的实际变量可以是不同类型
int n1 = 1;
int n2 = 3;
Console.WriteLine("数字{0},数字{1}", n1, n2); //1,3
对比：占位符按照挖坑的顺序输出,占位符从0开始
Console.WriteLine("数字{1},数字{0}", n1, n2); //3,1
string info="hehe"；
Console.WriteLine("{0},{1}",n1,info);

补充：ReadLine 接收控制台输入，并用字符串类型变量保存
string name=Console.ReadLine();
Console.WriteLine(name);

int类型加减乘除，结果也还是int类型

转义符

定义：\加上一个特殊意义字符，就组成了一个转义符
Console.WriteLine("这是一个需要转义的\"\"字符串");这样就能输出字符串中间的""
\t:tab
\b:相当于backspace，会删除\b前面一个字符，但是把\b放到字符串两端无效果
\n：换行，但是只是苹果系统和win控制台能识别，在windows系统(例如写文件，要使用\r\n);

原样输出：@
Console.WriteLine(@"f:\\c\test");
保留原格式输出：@
Console.WriteLine(@"晴天天气
            十分的好");

补充：char c=’\b’;
此时理论上应该是错误的，因为char只能放入一个字符，但是此时\b代表是backspace按键，就是个转义符，代表的是一个字符。

类型转换

显式类型转换：
double d=3.14;
int a=(int)d

隐式类型转换：不需要写代码，内部自动
int a=3;
double b=a;

利用占位符输出指定位数小数：
Console.WriteLine("{0:0.00}",10.0/3);//3.33
{0}，代表后面，第0个数字，另外，必须10.0不然结果是3.00.

Convert类型转换

上面的显式，隐式转换指的是可以类型兼容的类型，例如int,double
但是string转int或者其他，则需要借助，Convert工厂

do-while:循环体在do中

调试

F11：单步调试
F10：逐过程调试(方法上面使用)
断点调试

转换对比

使用Convert进行转换，如果失败会抛异常，内部调用int.Parese()，效率相比更低
使用int.Parse(字符串) ，失败会抛异常(同理有double.Parse等)
使用int.TryParse(字符串,out 变量) 失败返回false，成功返回true，不会抛异常，效率最高
int num=0;
bool b=int.TryParse("123",out num)
解析：b接收布尔型返回值，num接收转换成功后的数字（转换失败num输出0）

常量：const int num=10;

枚举

声明到命名空间下面，类的上面，表示这个命名空间，窦娥可以使用这个枚举

枚举类型默认可以跟int类型互相转换，枚举和int是兼容的
枚举默认是从0开始的，如果上面把man=2这样就是从2开始，可以强制转换之后，打印验证

结构

扩展：结构体内部字段可以使枚举类型，赋值时候，也是使用对应枚举类型赋值

数组

案例：冒泡排序
从大到小，或者从小到大顺序进行排列
int[] arr={5,8,1,3,9};
第一趟：5和8比，如果大于，换位置，然后继续和下一个(1)比较,以此类推，比较4次
第二趟：比较三次
每一个比较完成之后，会把最大或者最小的一个放到最后，以此类推

方法

Program.show();//Program代表类名，此时意思，类名调用方法
如果方法和main函数在一个类时候，可以省略类名，否则不可省略。

c#中基本数据类型，是值传递，相当于传递的是copy，不会影响原来的值
但是string特殊，是引用传递。

方法的高级参数

out参数

一个方法中，返回多个同一类型的值时候，可以返回一个数组。
但是如果返回多个不同类型的值得时候，返回数组就不行，可以使用out参数

ref参数

params可变参数

test2(1,2,3);//此处可以传递和params类型相同的参数，也可以直接传递一个数组(下面是int[])
public static void test2(params int[] num){}
注意：params修饰的参数数组，只能是参数列表的最后一个。
