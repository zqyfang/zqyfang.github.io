Html标签简介

发表于 2017-08-06 | 分类于 前端

一、属性值：标签内部，空格分隔

二、标签

(一)、文件标签

body有以下属性:

text:文本颜色
bgcoloc:背景色
background：背景图片

注解：background会覆盖背景色,而且注意font标签字体颜色使用color,而body字体使用text.

（二）、排版标签

属性：
size：宽度（不写单位默认px）
width：长度(可以写值，也可以写百分比)
align:对齐方式（不写默认根据父元素）

百分比和具体值区别：百分比随父元素变化而变化，例如缩放窗口。
注意此处，hr下面两项对齐方式已经是居左了，p对他们无影响了。

(三)、块标签

div:行级块标签,默认占据一行
span:行内块标签，不换行

注解：内部style是css属性

(四)、文字标签

font
属性：

color:颜色
 size:大小
（最大7号字体，在写大也只有7.最小值1，不写默认3）
 face:字体类型

(五)、清单标签（列表）

  • ul无序列表

    属性：type
    有square(方块) circle(圆圈) disc(默认)

  • oi有序列表

    属性：type
    写数字，就是1，2，3，形式
    写I，i 就是罗马字符
    写字母 就是a,b大小写这种
    start :首项开始位置

  • dl定义列表

    dt:标题
    dd:描述

(六)、图形标签

属性：align指的是图片和相邻文本的位置
middle指的就是居中，此时文本就在图片之后中央的位置。还可以bottom，top。
alt:网络不好，图片显示不出来的提示。
title：鼠标悬浮提示。

(七)、链接标签

target:_self(在本身打开) _blank(新建页面打开)

解析锚点，是根据name来确定的，最上面name=top,下面另一个a标签，href中的#代表本页面，然后跟上对应name值即可。

(八)、表格标签

caption：标题
th:表格头
tr:行
td:单元格
    td属性：
    rowspan：行合并
    colspan：列合并《注意这两者合并之后都要删除被合并的部分》
table属性：
    border:表格边框
    width：表格长度
    algin:表格对齐方式
    bgcolor:背景颜色

三、表单标签

  • form标签
    name:表单名称
    action：提交的地址
    method：提交方式

  • select标签
    name:名称（提交时候网址上面是这样的：name值=value值?…）
    option:代表一个选择项
    selected=”select”:默认被选择
    value:提交值

  • input标签
    type:根据不同值，显示不同功能的表单项。
    可以是：text,password,radio(单选),
    checkbox(复选),file(上传文件),
    button,submit,reset,
    image(图片按钮,也有src,alt等属性)，
    hidden(隐藏表单，服务端需要但是不需要用户看见)等值。

解析:单选时候通过相同name值实现互斥。复选的name值相同表示此时是同一组。value:默认值,比如当是radio(单选)时候，提交的就是value值。checked="check":默认选择。

  • textarea标签

属性：
    cols：列
    rows:行
    name:名称（提交之后：name值=textarea框中的值
    ）
注意：textarea没有value值，提交的值是框框中输入的内容。

四、框架标签(去掉body,直接用frameset)

right.html代码

解析：此处target指的就是跳转到哪，没有用_self这些属性，right对应上面
<frame src="http://www.baidu.com" name="right">中的name值right。
rows="120,*"：值得是上面120行，其他*表示，还可以这么写，rows="120,*,120"
这样的意思是上下各120，中间全部占有。

五、特殊标签

&nbsp:空格
&gt:大于
&lt:小于
&Reg:注册符号
&copy:版权符号

六、标签分类

div标签是个容器级标签，什么都可以放，甚至可以放div自己
span标签是个文本级标签，”小区域”，只能放置图片文字表单元素。
例如：p h ul dl ol div不可以放在span里面。
