Android 6.0权限

发表于 2017-08-06 | 分类于 Java

Android 6.0之前,权限在app安装时候，就直接默认授予，Android 6.0引入运行时权限，动态申请。

注意：同一组内的任何一个权限被授权了，其他权限也自动被授权。例如，一旦 READ_CALENDAR 被授权了，应用也有 WRITECALENDAR 权限了。
